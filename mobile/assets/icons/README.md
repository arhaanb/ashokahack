# Icons Folder

Place the following icons here:

## Navigation Icons
- `location-pin.png` - Location marker icon (green)
- `notification.png` - Bell notification icon
- `dropdown-arrow.png` - Small dropdown arrow
- `search.png` - Search magnifying glass icon
- `filter.png` - Filter/sliders icon
- `heart-outline.png` - Empty heart for favorites
- `heart-filled.png` - Filled heart for favorites
- `star.png` - Star rating icon

## Tab Bar Icons
- `explore.png` - Shopping bag icon for Explore tab
- `explore-active.png` - Active state (green) shopping bag
- `favourites.png` - Heart icon for Favourites tab
- `favourites-active.png` - Active state heart
- `profile.png` - Person icon for Profile tab
- `profile-active.png` - Active state person icon

All icons should be PNG with transparent background.
Recommended sizes: 24x24 or 32x32 pixels (provide @2x and @3x versions).
