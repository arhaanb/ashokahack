# Restaurant Images Folder

Place restaurant/food images here.

## Naming Convention
Use restaurant ID or slug as filename:
- `bahadur-bhandar.jpg`
- `mithas-restaurant.jpg`
- `spice-symphony.jpg`
- `curry-leaf.jpg`
- `sagar-ratna.jpg`
- `tandoori-tales.jpg`

## Image Specifications
- Size: 400x200 pixels minimum (will be cropped to fit card)
- Format: JPG or PNG
- Style: Restaurant interior/food photos
- Aspect ratio: ~2:1 (width:height)

## Placeholder
A default placeholder image should be named:
- `placeholder.jpg` - Default image when restaurant image is not available
